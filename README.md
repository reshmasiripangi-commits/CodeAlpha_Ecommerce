# 🛒 E-Commerce Store

A fully functional E-Commerce web application built using Django. Users can browse products, view product details, add items to cart, place orders, and manage their accounts.

---

## Features

### User Management
- User Registration
- User Login & Logout
- Authentication System

### Product Management
- View Products
- Product Detail Pages
- Product Images
- Category-wise Organization

### Shopping Features
- Add to Cart
- View Cart
- Remove Items from Cart

### Order Management
- Place Orders

### Media Support
- Product Image Uploads
- Media File Handling

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/ecommerce_project.git
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install django
```

Or:

```bash
pip install -r requirements.txt
```

### Apply Migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### Create Superuser

```bash
python manage.py createsuperuser
```

### Run Server

```bash
python manage.py runserver
```

Visit:

```text
http://127.0.0.1:8000/
```

---

## Admin Panel

Access Django Admin:

```text
http://127.0.0.1:8000/admin/
```

Admin can:

- Add Products
- Upload Product Images
- Manage Users
